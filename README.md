# Responsive Dashboard UI

A modern, responsive dashboard interface created using HTML and CSS.  
Includes sections for activities, dashboard data, user interactions, schedules, and contact details with an embedded map.

### Highlights
- Fully responsive (desktop/Tablet/mobile)
- Clean sidebar navigation
- Data cards and activity graph
- User statistics table
- Scrollable meeting schedule
- Feedback form

### Usage
Open the HTML file in a browser. No installation required.
